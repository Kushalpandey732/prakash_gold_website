# Prakash Gold Website

Modern multipage jewelry website built with React (frontend) and Node.js/Express (backend).

## Pages

- Home
- About the Founders
- About the Company
- Contact (Dubai map + enquiry form)
- Footer with social media links

## Run Locally

### Backend

1. `cd backend`
2. `copy .env.example .env`
3. Update SMTP values in `.env`
4. `npm install`
5. `npm run dev`

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`

Frontend runs on Vite default (`http://localhost:5173`) and backend on `http://localhost:5000`.
